html < input> Element

< input> element can be displayed in different ways.

input type="text" Displayes a single-line text input field.
input type ="radio" Displayes a radio button for selecting one in different choices.
input type ="checkboxe". Displayes a checkbox for selecting zero or none in different choices
input type="submit" Displayes a submit button
input type="button" Displayes a button to click.

target attribute specifies where to display the response after submiting the form.
_blank The response is displayed in a new window or tab.
_self the response is displayed in the current window.
There are other's target attributes.
Syntax
